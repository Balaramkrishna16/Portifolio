Home security system

The IoT-Based Home Security System is a smart solution designed to improve home safety using Arduino, sensors, and an IoT platform. It detects motion near the entrance, automatically activates a solenoid lock, and sends real-time alerts to the user via Firebase. With a simple interface and reliable performance, the system offers efficient security automation. Future upgrades include AI-based threat detection, cloud integration, and energy-efficient features.

Farm2Home

Farm2Home is a web-based platform designed to connect vendors and suppliers, ensuring farmers get fair prices for their produce. It features separate panels for vendors and suppliers to manage products, orders, and pricing efficiently. The front end is built using HTML, CSS, and JavaScript, while Firebase handles user authentication and database management. This project promotes transparency and supports fair trade within the agricultural market.
